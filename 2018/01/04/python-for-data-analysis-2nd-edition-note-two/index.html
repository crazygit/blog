合并数据集

数据合并操作主要有以下几个方法:

  • pandas.merge: 基于key来合并row，类似关系型数据库的JOIN操作
  • pandas.concat: 在轴上合并数据，类似stack操作
  • combine_first: 合并重复的数据和填充确实的数据
